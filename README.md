# css-filters

Apply Instagram-style CSSgram filters to your own local images and download the result as a PNG.

## What this repo does

- Loads CSSgram filter classes from `cssGram.html`
- Lets you choose a local image from your machine
- Previews all available Instagram-style filter variants
- Exports the selected filtered preview to a downloadable PNG

## Quick start

1. Clone the repo:

   ```bash
   git clone https://github.com/ironicbuddha/css-filters.git
   cd css-filters
   ```

2. Open `cssGram.html` in your browser.
3. In **Local Image**, select an image from your computer.
4. Click any filter card to open the large preview.
5. Click **Download PNG** to save the filtered image.

## Notes

- URL-based images still work, but local images are the most reliable for export.
- The app is static HTML/CSS/JS and requires no build step.
